> 难度：简单
- 思路：双指针，快慢指针


> 题目
给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

> 代码

```cpp
class Solution {
public:
    ListNode* removeNthFromEnd(ListNode* head, int n) {
        if(head == nullptr) return nullptr;
        ListNode* fast,*slow;
        fast = head;
        slow = head;
        while(n > 0){
            fast = fast->next;
            --n;
        }
        // 删除的是第一个
        if(fast == nullptr){
            return head->next;
        }
        while(fast != nullptr && fast->next != nullptr){
            fast = fast->next;
            slow = slow->next;
        }
        slow->next = slow->next->next;
        return head;
    }
};

```
