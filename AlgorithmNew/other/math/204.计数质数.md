> 难度：简单
- 思路：质数相关的算法
> 题目
```
统计所有小于非负整数 n 的质数的数量。

示例 1：
输入：n = 10
输出：4
解释：小于 10 的质数一共有 4 个, 它们是 2, 3, 5, 7 。

示例 2：
输入：n = 0
输出：0

```

> 代码

```cpp
class Solution {
public:
    int countPrimes(int n) {
        vector<bool> isPrime(n+1,true);
        // 最大数n的分解，最多也只需要看到sqrt(n)，所以遍历到sqrt(n)，就全部被填充排除了
        for(int i  = 2; i*i < n; ++i){
            if(isPrime[i])
                for(int j = i*2; j < n; j+=i){
                    isPrime[j] = false;
                }
        }

        int res = 0;
        for(int i = 2; i < n; ++i){
            if(isPrime[i])
                ++res;
        }
        return res;
    }

};
```
