# 快捷键
 >https://blog.csdn.net/ruglcc/article/details/7805212
 
| 快捷键           | 功能         |
| ---------------- | ------------ |
| Ctrl + Shift + E | 垂直分割窗口 |
| Ctrl + Shift + O | 水平分割窗口 |
|  Ctrl + Shift + N | 在分割的各个窗口自由切换  |
| Ctrl + Shift + F  | 查找 |
| Ctrl + Tab | 终端切换 |
| Alt + 方向键 | 终端切换 |
| Ctrl + '+'/'-' | 字体改变 |
| Ctrl + Shift + C/V | 复制/粘贴 |
|  Ctrl + Shift + W  | 关闭当前终端 | 
| Ctrl + Shift + X | 最大化当前窗口 |
| Ctrl + Shift + T | 打开一个新标签 |
| Ctrl + Shift + Q  | 关闭Terninator | 
| f11 | 全屏 |
